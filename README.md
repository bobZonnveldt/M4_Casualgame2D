# M4_Casualgame2D
# Gekozen uitdaging
## Ik kies:  Extra Fun

# Core mechanic
## Mijn core mechanic is dat de speler balletjes schiet om balletjes te laten verdwijnen

# Input
## De speler doet dit door te clicken en te richten

# Doel
## De speler probeert zo lang mogelijk to overleven of alle balletjes weg halen

# Easy to learn
## De speler begrijpt snel dat ze moeten schieten

# Hard to master
## Het wordt uitdagend door dat er de hele tijd balletjes bij komen

# Waarom blijft dit leuk?
## Deze actie blijft leuk om te herhalen omdat het leuk is om groote combos te maken

# Juiciness
## Ik maak deze actie satisfying door de combos en het weghalen goed en lekker te laten werken
