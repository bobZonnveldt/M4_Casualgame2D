# M4_Casualgame2D
# Gekozen uitdaging
## Ik kies:  Fun

# Core mechanic
## Mijn core mechanic is dat de speler balletjes schiet om balletjes te laten verdwijnen

# Input
## De speler doet dit door te clicken en te richten

# Doel
## De speler probeert zo lang mogelijk to overleven of alle balletjes weg halen

# Easy to learn
## De speler begrijpt snel dat ze moeten schieten

# Hard to master
## Het wordt uitdagend door dat er de hele tijd balletjes bij komen

# Waarom blijft dit leuk?
## Deze actie blijft leuk om te herhalen omdat het leuk is om groote combos te maken

# Juiciness
## Ik maak deze actie satisfying door de combos en het weghalen goed en lekker te laten werken


# input-actie-gevolg-feedback
## je schiet een bubble/bal naar een andere van dezelfde kleur-de actie is een een combo to halen door 3 of meer bubbles van dezelfde kleur op een rij te hebben-het gevolg is dat de bubbles weg gaan-feedback is een score en een geluidje 

# wat hoort de speler
## de speler hoort een bubble poppen als ze een combo krijgen en als ze geen combo krijgen maar gewoon een bubble tegen een andere bubble dan is het een slimy bubble sound

# wat verandert er in score/timer/progessie
## als je een combo haalt dan krijg je score erbij en elke 5 keer dat je schiet komen er weer extra ballen bij 

# wat beweegt of animeert
## de bubbles 

# wat maakt de actie satisfying
## de sounds en extra screen effects zoals screen shake en een outline en de animaties

# wanneer start de actie
## wanneer de speler de bal schietschiet/krijgt

# wanneer is de actie gelukt
## als de speler de balleen naast elkaar krijgt schiet/krijgt

# wat verandert er
## als er 3 ballen van dezelfde kleur naast elkaar wordt geschoten dan gaan ze kapot en verandert de map dus 

# welke variable heb je nodig 
## int string float

# welke feedback krijgt de speler
## screen shake en text in beeld
